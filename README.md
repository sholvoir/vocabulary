# Vocabulary

词书生成工具，使用TypeScript编写, 运行环境为Deno。

if you want to build all data by youself, you need run it use [Deno](https://deno.land/).  
You can download **vocabulary.txt** directly.

How to run:
```bash
deno task run[ -d][ name]
deno task release
```
-d: debug.  
name: run "name" section only.
